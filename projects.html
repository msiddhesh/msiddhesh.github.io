Welcome to Projects

Air Pollution Dashboard

An interactive shiny dashboard to help you monitor ambient air quality in Delhi.

A real-time dashboard that hosts choropleth maps built using inverse distance weighted interpolation of monthly average data, interactive color-coded time series plots. The dashboard is hosted using R Shiny available here.




DelhiAirQuality Twitter Bot

Automated updates on air pollutants in Delhi.

An automated twitter bot tweeting the last 24-hr trends for four pollutant parameter PM2.5, PM10, NH3, SO2 and AQI for 24 ambient air quality stations. This bot is built on top of the shiny dashboard displaying color-coded time-series plots of real-time air quality data available here.




BestModelSubset.jl

A julia package to implement model selection algorithms on basic regression models.

It basically implements model selection algorithms to simple regression models. Currently, it offers best subset selection, forward step-wise selection and backward step-wise selection to linear and logistic regression models.
